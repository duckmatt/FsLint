# PatternMatchClausesOnNewLine (FL0004)

## Cause

All pattern match clauses should be on their own line.

## Rationale

Readability.

## How To Fix

Place each pattern match clause on its own line.

## Rule Settings

	[lang=javascript]
    {
        "patternMatchClausesOnNewLine": { 
            "enabled": false
        }
    }
