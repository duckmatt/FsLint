# MaxLinesInFile (FL0062)

## Cause

More than a configurable number of lines were found in a file.

## Rationale

Too many lines in a file indicate it's becoming too complex.

## How To Fix

Refactor to extract code out into another file.

## Rule Settings

	[lang=javascript]
    {
        "maxLinesInFile": { 
            "enabled": true,
            "maxLinesInFile": 1000
        }
    }
