# PatternMatchOrClausesOnNewLine (FL0005)

## Cause

All pattern match "or" clauses should be on their own line.

## Rationale

Readability.

## How To Fix

Place each pattern match "or" clause on its own line.

## Rule Settings

    {
        "patternMatchOrClausesOnNewLine": {
            "enabled": false
        }
    }
