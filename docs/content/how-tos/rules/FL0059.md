# Indentation (FL0059)

## Cause

Inconsistent indentation in file.

## Rationale

Readability.

## How To Fix

Update file to use consistent amount of spaces for indentation.

## Rule Settings

Uses the `numIndentationSpaces` global setting.

    {
        "indentation": {
            "enabled": false
        }
    }
